🏛️ Campus Navigation and Management System

 📝 Description
A Windows Forms application for managing university operations including student, course, and teacher management with campus navigation features.

🚀 Features
- **Admin Portal**: Manage Students, Courses, Teachers, Approve Admins
- **Student Portal**: View Campus Map, Find Route, View Timetable, Fresher Tips, FAQs
- **Role-based Authentication**: Admin and Student roles
- **Database Integration**: MySQL with CRUD operations

 🛠️ Technologies
- C# Windows Forms
- MySQL Database
## ARCHITECTURE
- 3-Tier Architecture (UI, BL, DL)
- OOP Principles (Encapsulation, Inheritance, Polymorphism, Abstraction)
